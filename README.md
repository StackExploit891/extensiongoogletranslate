## 📖 Overview

The **Google Translate Extension** is a free browser add-on powered by Google's neural machine translation engine. It detects when you visit a page written in a foreign language and offers an instant, full-page translation — right inside your browser tab.

Supports translation between **133+ languages** with a single click.

## 🚀 Installation

### Google Chrome / Microsoft Edge

1. Open the [Chrome Web Store](https://chrome.google.com/webstore/detail/google-translate/aapbdbdomjkkjkaonfhkkikfgjllcleb)
2. Click **"Add to Chrome"** (or **"Add to Edge"**)
3. Confirm by clicking **"Add Extension"** in the dialog
4. The 🌐 Google Translate icon will appear in your browser toolbar

### Mozilla Firefox

1. Visit the [Firefox Add-ons page](https://addons.mozilla.org/en-US/firefox/addon/to-google-translate/)
2. Click **"Add to Firefox"**
3. Grant the required permissions when prompted
4. Pin the extension to your toolbar for quick access

## 🛠️ How to Use

### Translate a Full Page

```
1. Navigate to any foreign-language webpage
2. A prompt bar will appear at the top of the browser
3. Click "Translate" — the entire page translates instantly
4. Click "Show original" to revert at any time
```

### Translate Selected Text

```
1. Highlight any word, phrase, or paragraph on a webpage
2. Right-click and select "Translate '[selected text]'"
   — OR —
   A small Google Translate icon will pop up near your selection
3. A tooltip shows the translation immediately
```

### Use the Extension Icon

```
1. Click the 🌐 icon in your browser toolbar
2. Choose your target language from the dropdown
3. Click "Translate this page"
```

## ⚙️ Settings & Preferences

Access extension settings by:
- Right-clicking the toolbar icon → **"Options"**
- Or navigating to `chrome://extensions` → Google Translate → **"Details"** → **"Extension options"**

### Available Options

- **Never translate [language]** — Skip auto-detection for specific languages
- **Always translate [language]** — Auto-translate certain languages without prompting
- **Never translate [domain]** — Exclude specific websites from translation prompts
- **Default target language** — Set your preferred translation language globally

## 🔒 Permissions & Privacy

The extension requests the following browser permissions:

| Permission | Reason |
|---|---|
| `Read and change site data` | Required to inject translated content into the page |
| `Access browser tabs` | Needed to detect the active tab's URL and language |
| `Storage` | Saves your language preferences locally |

> ℹ️ **Note:** Text is sent to Google's servers for translation. Do not use on pages containing sensitive personal or confidential information. Review [Google's Privacy Policy](https://policies.google.com/privacy) for full details.

## ❓ FAQ

**Q: Is the extension free?**  
A: Yes, the Google Translate Extension is completely free to install and use.

**Q: Does it work offline?**  
A: No, an internet connection is required as translations are processed on Google's servers.

**Q: Can I translate PDFs opened in the browser?**  
A: Yes, Chrome and Edge support in-browser PDF translation via the extension.

**Q: Why does the translated text look misaligned on some sites?**  
A: Translated text may differ in length from the original, causing minor layout shifts. This is expected behavior.

**Q: How accurate is the translation?**  
A: Google Translate uses neural machine translation (NMT) and is highly accurate for most major languages. Technical, legal, or highly idiomatic content may still require human review.
